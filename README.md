نبذة عن التطبيق
تطبيق السهم للتسكين هو منصة متكاملة تسعى لتوفير حلول سكنية مناسبة للطلاب بطريقة سهلة وآمنة، بواجهة عربية سلسة ومميزات متعددة تلبي احتياجات الطلاب في مختلف المراحل الدراسية.
